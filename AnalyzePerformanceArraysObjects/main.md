# Analyzing Performance of Arrays and Objects

## Big O of Objects
Unordered list
Insertion - O(1)
Removal - O(1)
Searching - O(n)
Access - O(1)

Object.keys() - O(n)
Object.values() - O(n)
Object.entries() - O(n)
hasOwnProperty() - O(1)
